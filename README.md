<h1 align="center">
   DogsAPI
</h1>

Projeto em Kotlin utilizando os seguintes recursos da linguagem:

- Arquitetura MVVM
- Navigation
- Room
- Data Binding
- AndroidX
- KTX extensions
- Lifecycles
- Live Data
- ViewModel
- Notifications
- Permissions
- Sharing
- Preferences
- Fragments
- Layouts
- Palette
- Multidex
- RxJava
- Retrofit
